# ClientPro Hub v1.8 Combined CRM Project Commerce

This package combines v1.7 and v1.8 scope.

## Combined Features
- All modules routing from Dashboard to Settings
- CRM ↔ Project relationship
- Customer profile with linked projects and commerce records
- Project creation requires CRM customer selection
- Commerce Core functional workflow:
  - Create Quotation
  - Approve Quotation + Create Delivery Order
  - Create Invoice from Quotation
  - Record Payment against Invoice
  - Auto-generate Official Receipt
  - Payment updates invoice paid/balance/status
  - PDF preview placeholder for quotation, delivery order, invoice and receipt
- Website Builder linked to CRM customer/project
- Support tickets linked to CRM customer/project
- Assets/Documents linked to CRM customer
- Reports include CRM relationship summary

Recommended commit message:
`Add v1.8 combined CRM project and commerce workflow`
